# Terraform + Ansible Infrastructure Automation

## 📌 Project Overview

This project automates AWS infrastructure provisioning using Terraform and configures EC2 instances using Ansible.

## 🚀 Technologies Used

- AWS EC2
- Terraform
- Ansible
- Java 21
- Docker
- Jenkins
- Ubuntu

## 📂 Project Structure

terraform/
ansible/
roles/
inventory
site.yml

## ⚙️ Workflow

1. Create EC2 instance using Terraform
2. Generate Inventory
3. Run Ansible Playbook
4. Install Java
5. Install Docker
6. Install Jenkins
